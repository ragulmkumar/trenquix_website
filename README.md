# Trenquix Website

Official company website for **Trenquix** - a premium web development studio.

## Tech Stack

- Next.js (App Router)
- React
- Tailwind CSS
- JavaScript (no TypeScript)

## Project Structure

- `src/app/` - app routes, layout, global styles
- `src/app/components/` - all landing page sections
- `public/` - static assets (including favicon/logo)

## Getting Started

Install dependencies:

```bash
npm install
```

Run development server:

```bash
npm run dev
```

Open: http://localhost:3000

## Production

Build:

```bash
npm run build
```

Start:

```bash
npm run start
```

## Lint

```bash
npm run lint
```

## Deploy (Vercel)

1. Push this repository to GitHub.
2. Import the project in Vercel.
3. Framework preset: **Next.js**.
4. Deploy.

No extra environment variables are required for the current version.
